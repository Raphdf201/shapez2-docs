# Setup

# 1.1 : Modding environment

First, setup the shapez shifter API ([official instructions](https://tobspr-games.notion.site/shapez2-modding-documentation#2583c9e752e080728e40eae35a163fce)).
Make sure that the `SPZ2_SHIFTER` environment variable is set properly by executing the following command in a terminal : `echo %SPZ2_SHIFTER%` on windows, or `echo $SPZ2_SHIFTER` on linux/mac.

# 1.2 : Project

I made a simple [project generator](https://generator.shapez2.raphdf201.net/) to get started easily.
